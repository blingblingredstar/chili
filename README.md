# antd library

基于 ant-design 封装的组件。

## usage 使用

```bash
$ npm install chili -S
```

## contribution 贡献开发

Install dependencies,

```bash
$ npm i
```

Start the dev server,

```bash
$ npm start
```

Build documentation,

```bash
$ npm run docs:build
```

Build library via `father-build`,

```bash
$ npm run build
```
